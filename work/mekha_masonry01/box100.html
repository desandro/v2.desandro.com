Box100 example

This example shows:

  1. Masonry 0.8 can handle a multiple column widths. This example has 5 different widths (col1, col2, col3, col4 col5)
  2. No height is specified on the image tags, but masonry is not called until all the images are loaded. This is done be calling masonry inside $(window).load() instead of $(document).ready() 
    $(window).load(function(){
	$('#primary').masonry({columnWidth: 100, itemSelector: '.box' });	
});
    Try a full fresh (Shift + Refresh) to see how masonry only is activated until after all the images are loaded.
  3. How using more column widths will produce more gaps
  4. At the bottom you can dynamically add more content to the Masonry layout by clicking the button.
